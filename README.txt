
BLOOD DONATION MANAGEMENT SYSTEM - README

📁 How to Run:
1. Copy this folder to htdocs/ if using XAMPP.
2. Start Apache and MySQL from XAMPP control panel.
3. Import 'blood_donation.sql' into phpMyAdmin to create the database.
4. Visit: http://localhost/blood_donation_complete/index.php
